Төмендегі мысалдар әртүрлі визуализация кітапханаларын қай жағдайда қолдануға болатынын көрсетеді. Код үлгісінің астында сол графиктің ықшам көрінісі берілген.

Matplotlib

Сызықтық график

import matplotlib.pyplot as plt

months = ['Қаң', 'Ақп', 'Нау', 'Сәу']
sales = [12, 18, 9, 24]

plt.plot(months, sales, marker='o')
plt.title('Айлық сатылым')
plt.xlabel('Ай')
plt.ylabel('Сатылым')
plt.grid(True)
plt.show()
Қай дерекке қолайлы: уақыт бойынша өзгеріс, динамика, тренд көрсету.
Seaborn

Бағандық диаграмма

import seaborn as sns
import matplotlib.pyplot as plt

groups = ['A', 'B', 'C', 'D']
scores = [78, 85, 69, 92]

sns.barplot(x=groups, y=scores)
plt.title('Топтар нәтижесі')
plt.xlabel('Топ')
plt.ylabel('Ұпай')
plt.show()
Қай дерекке қолайлы: категорияларды салыстыру, топтар нәтижесін көру.
Plotly

Нүктелік диаграмма

import plotly.express as px

hours = [1, 2, 3, 4, 5]
scores = [55, 62, 70, 76, 88]

fig = px.scatter(x=hours, y=scores,
                 labels={'x':'Оқу сағаты', 'y':'Ұпай'},
                 title='Оқу уақыты мен нәтиже')
fig.show()
Қай дерекке қолайлы: екі көрсеткіш арасындағы байланысты анықтау.
Pandas plotting

Дөңгелек диаграмма

import pandas as pd
import matplotlib.pyplot as plt

data = pd.Series([40, 25, 20, 15],
                 index=['Код', 'Тест', 'Debug', 'Рефлексия'])

data.plot(kind='pie', autopct='%1.0f%%')
plt.ylabel('')
plt.title('Тапсырма түрлері')
plt.show()
Қай дерекке қолайлы: бүтіннің ішіндегі үлестерді көрсету.
